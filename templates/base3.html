We make use of three algorithms that use extremely different techniques to find the outliers in the given dataset. An outlier is defined as Type 1 if it is detected by only of the algorithms. Similarly it is labelled as Type 2 when it is detected by 2 of the algorithms and Type 3 when all of the algorithms state that the data point is an outlier. Along with this the user gets the option to get a list of indices at which the Type 1, Type 2, and Type 3 outliers are found. Please ensure that all the coulmn consist of only numerical values.
