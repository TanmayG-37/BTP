Here we implemented three multivariate outlier detection algorithms, of which HBOS and Isolation Forest are the existing best practice methods. FIND is a novel algorithm that scales large datasets.


HBOS is a histogram-based outlier detection method that does not require prior training (Goldstein et al., 2012)


Isolation forest works on the principle of the decision tree algorithm. It isolates the outliers by randomly selecting a feature from the given set of features and then randomly selecting a split value between the maximum and minimum values of the selected feature. This random partitioning of features will produce smaller paths in trees for the anomalous data values and distinguish them from the normal set of the data (Liu et al., 2008).


Unlike HBOS and Isolation Forest, FIND uses hash families for binary encoding of high-dimensional data points. FIND is faster than HBOS and Isolation Forest and GPU friendly.
