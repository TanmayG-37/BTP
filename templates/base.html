Summary

Outliers arise in diverse disciplines. Some common applications of outlier detection include detecting banking and credit card frauds, manufacturing defects, network intrusion, etc. Outlier detection algorithms could be univariate or multivariate. Multivariate methods are typically computationally expensive. To circumvent this, we developed FIND that uses the power of Big Data algorithms such as hashing. It can be implemented on GPU.
