About Us

Welcome to Outlier Detection! One can use our API in order to detect outliers in their given dataset. Our approach can be applied in various fields of study as it also gives the outliers a type thereby indicating the degree to which it is distinct from the given dataset. The user is expected to provide a CSV file with also the attributes and our algorithm will return the raw numbers as well as a visual representation of the Outliers present in the dataset.
